### 🔴 ภาพรวมระบบ

**I AM AUDIT** — ระบบตรวจสอบภายในสำหรับองค์กรปกครองส่วนท้องถิ่น (อปท.) แบบ **Single-File Web App** — ไม่ต้องติดตั้ง ไม่ต้องมี server ทำงานได้ทันทีใน browser

**ธีม:** แดง · ดำ · ขาว | **ภาษา:** ไทย | **Storage:** localStorage

---

### 🏠 หน้า Landing

เปิดระบบมาจะพบหน้า Landing แสดง 3 ภารกิจ:

| ภารกิจ | สถานะ |
|--------|-------|
| 🔍 งานตรวจสอบภายใน (Internal Audit) | ใช้งานได้เต็มรูปแบบ |
| 🛡️ งานควบคุมภายใน (Internal Control) | เร็ว ๆ นี้ |
| ⬡ งานบริหารความเสี่ยง (Enterprise Risk Management) | เร็ว ๆ นี้ |

### 🔐 ระบบสมาชิก (Login) — 2 สิทธิ์การใช้งาน

| สิทธิ์ | เข้าถึง | ต้อง Login |
|--------|---------|-----------|
| **ผู้ใช้งานระบบ** (เจ้าหน้าที่ตรวจสอบ) | ดู + แก้ไข/บันทึก/ลบ ได้ทุกโมดูล | ✅ ต้อง Login |
| **ผู้ใช้งานทั่วไป** | ดูข้อมูลและดาวน์โหลดเอกสาร/รายงานได้ | ❌ ไม่ต้อง Login |

- บัญชีเริ่มต้นถูกกำหนดไว้ในโค้ด (ตัวแปร `DEFAULT_ACCOUNTS`) — **แนะนำให้เปลี่ยนรหัสผ่านทันทีหลัง deploy** ผ่านปุ่ม "🔑 เปลี่ยนชื่อผู้ใช้/รหัสผ่าน" ในแถบเมนูด้านซ้ายหลัง Login
- ⚠️ **ข้อควรทราบด้านความปลอดภัย:** ระบบนี้เป็น static web app ฝั่ง client ล้วน ไม่มี server/database ตรวจสอบรหัสผ่านเป็นการเทียบค่าใน `localStorage` ของเบราว์เซอร์เท่านั้น เหมาะสำหรับกันการแก้ไขข้อมูลโดยไม่ตั้งใจของผู้เยี่ยมชมทั่วไป **ไม่ใช่ระบบรักษาความปลอดภัยระดับสูง** ไม่ควรใช้เก็บข้อมูลลับ/ข้อมูลส่วนบุคคลที่อ่อนไหวโดยไม่มีมาตรการเพิ่มเติม

### 🗂️ ศูนย์เอกสาร (Document Center)

- **อัพโหลด** ลากไฟล์มาวาง (Drag & Drop) หรือคลิกเลือก
- **หมวดหมู่** กฎบัตร / ความเสี่ยง / แผน / Fieldwork / รายงาน / ติดตามผล / กฎหมาย / อื่น ๆ
- **ค้นหา** แบบ Full-text ทั้งชื่อไฟล์ คำอธิบาย แท็ก หมวดหมู่ กอง
- **ตัวกรอง** หมวดหมู่ / สำนัก-กอง / ปีงบประมาณ
- **เรียงลำดับ** วันที่ / ชื่อ / ขนาด
- **มุมมอง** Grid (thumbnail) และ List (ตาราง)
- **ดาวน์โหลด** กลับมาได้ทุกไฟล์ที่อัพโหลด — ใช้ได้แม้ไม่ Login
- **Preview** รูปภาพ inline ใน modal
- **แท็ก** กำหนดแท็กได้อิสระ
- รองรับ PDF, Word, Excel, PowerPoint, รูปภาพ, CSV, TXT, ZIP
